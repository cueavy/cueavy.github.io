langful-0.19

自制python本地化模块

langful

安装

可以使用pip直接安装pip install langful 或 pip3 install langful

这是其Github链接
这是其pypi链接

使用

定义 langful 类

import langful

lang = langful.lang()

参数解释

名称 解释 类型
lang_dir 本地化文件存放目录 str
default_lang 默认使用的语言 str
file_suffix 本地化文件的后缀.json或.lang str
change 使用什么字符来做替换 str

属性解释

名称 解释 类型
lang_dir 翻译文件的存放目录 str
default_lang 默认使用的翻译 str
file_suffix 本地化文件的后缀.json或.lang str
default_locale 默认语言 str
use_locale 选定的语言 str
lang_file 选择使用的语言文件 str
lang_file_list 所有能找到的语言文件 list
lang_str_list 所有能找到的语言文件的名字 list
language_dict 加载所有可以读取的语言文件 dict
language 加载需要的语言文件 dict
change 使用什么字符来做替换 str

函数解释

注:每个函数中的lang_str都可以不填

名称 解释 返回值
get() 获取指定键的值 str
set() 设置指定键的值 -
replace() 给一个字符串 返回替换好翻译的字符串 str

本地化文件编写方式

推荐使用json文件

遵循 json 语法 即可

示例

{
    "_" : "可以写些说明信息" ,
    "hi" : "你好" ,
    "welcome" : "欢迎"
}

文件默认储存在 \lang\{文件名}.json

语言代码表 参考

注意 应使用全小写

调用方法

get()

获取指定键的值

名称 解释 类型
key 字典的键 str
lang_str 本地化名称 如 zh_cn en_us 等等 str

set()

设置指定键的值

名称 解释 类型
key 字典的键 str
value 字典的值 str
lang_str 本地化名称 如 zh_cn en_us 等等 str

replace()

给一个字符串 返回替换好翻译的字符串

名称 解释 类型
args 字符串 str
lang_str 本地化名称 如 zh_cn en_us 等等 str
change 使用什么字符来做替换 str

lang_str与change均可不填

其它功能

if_then()

如果某个特定项的布尔值为真或假 那么替换为对应的字符串或列表进行替换

名称 解释 类型
args 字符串 str
replace_list 给一个字符串或列表 来替换字符串 list or str
change 选择用什么符号做替换 str
else_replace 如果为否 那么用这个来做替换 str

返回值类型 srt
